在 Java 中，成员变量（或实例变量）是类中的属性，它们用于存储对象的状态。每个实例变量在创建对象时拥有独立的值，属于对象本身。

成员变量的示例

在 Animal 类中，name 和 age 就是成员变量。每个 Animal 对象都会有自己独立的 name 和 age 值。

使用示例

通过创建 Dog 对象，可以看到 Dog 类继承了 Animal 的成员变量 name 和 age，并增加了自己的成员变量 breed。使用 displayInfo 方法可以展示所有的信息。

输出结果

解释

  • 继承：Dog 类继承了 Animal 的 name 和 age 成员变量，因此可以直接使用这些变量。
  • 构造方法：Dog 构造方法通过 super(name, age); 调用父类构造方法来初始化 name 和 age，再初始化自己的成员变量 breed。
  • 重写方法：在 Dog 中重写了 displayInfo() 方法，调用 super.displayInfo(); 先显示 name 和 age，然后增加 breed 的输出。

这样，子类不仅继承了父类的成员变量，还可以灵活地在子类中扩展和重写父类的行为。
